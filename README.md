## Как запустить

```sh
# установите зависимости
npm ci

# соберите клиентский код приложения
npm run build

# запустите сервер
npm start

# для запуска с bug id меньше 4 необходимо дополнительно перезапускать сервер
BUG_ID=# npm run start

# для остальных будет достаточно (но это не точно)
BUG_ID=# npm run test
```
